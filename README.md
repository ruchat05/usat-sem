# USAT SEM — Best Prepared Meal Delivery Services

A pixel-faithful implementation of the [Figma design](https://www.figma.com/design/kpTXdkTQrLhzX1xipD95dm/USAT---SEM?node-id=6-317) for the USA TODAY CO. prepared meal delivery comparison landing page.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 19 + TypeScript
- Vite
- Plain CSS (design tokens from Figma)

## Page Sections

- Header with USA TODAY CO. logo
- Hero with advertiser disclosure
- Top 3 comparison cards (Factor, Home Chef, Cook Unity)
- Full ranking list (10 services)
- Sidebar with guides and reviews
- SEO content sections with promo banners
- Footer
